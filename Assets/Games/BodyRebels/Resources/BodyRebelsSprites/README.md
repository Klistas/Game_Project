# Body Rebels Prototype Sprite Slots

이 폴더에 PNG/JPG 이미지를 넣으면 `BodyRebelsPrototype`이 실행 시 자동으로 불러옵니다.
파일명은 아래 이름을 그대로 쓰고, 확장자는 Unity가 인식하는 이미지 형식이면 됩니다.

## 장소 배경

- `venue_interview.png` - 면접실 배경
- `venue_date.png` - 소개팅 카페 배경
- `venue_store.png` - 편의점 배경
- `venue_dinner.png` - 회식 자리 배경

권장: 16:9 배경 전용 이미지, 1280x720 또는 1920x1080. UI 글자나 버튼은 이미지 안에 넣지 않는 편이 나중에 교체하기 쉽습니다.

## 주인공/상대 캐릭터

- `avatar_body.png`
- `avatar_mouth.png`
- `avatar_left_hand.png`
- `avatar_right_hand.png`
- `avatar_legs.png`
- `npc_body.png`

- `npc_eyes.png`

권장: 투명 배경 PNG, 512x512 또는 1024x1024. 몸 부위별로 따로 잘라두면 반응 애니메이션을 유지한 채 교체할 수 있습니다.

## 연출/소품

- `reaction_burst.png` - 폭주/당황 효과
- `exit_arrow.png` - 도망 본능 표시
- `table_prop.png` - 테이블/카운터
- `left_prop.png` - 왼쪽 배경 소품
- `right_prop.png` - 오른쪽 배경 소품

권장: 투명 배경 PNG. 없으면 기존 단색 플레이스홀더가 계속 표시됩니다.

## 넣는 방법

1. 이미지를 이 폴더에 복사합니다.
2. 위 파일명 중 하나로 이름을 맞춥니다.
3. Unity가 자동으로 import할 때까지 기다립니다.
4. Play를 누르면 해당 이미지가 바로 적용됩니다.

채팅에 이미지를 첨부해주면 제가 이 폴더로 복사하고, 맞는 슬롯 이름으로 정리해줄 수 있습니다.
