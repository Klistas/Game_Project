# Intended Feature AI Asset Prompts

목표: 삭제 대상 버그가 QA 패치를 악용해 세계 법칙을 망가뜨리고 “의도된 기능”으로 승격되는 2D 사이드뷰 시스템 코미디 느낌을 만든다.
생성 이미지는 `Assets/Games/IntendedFeature/Resources/IntendedFeatureSprites`에 파일명 그대로 넣는다.

## 공통 스타일

```text
Korean indie 2D side-view game asset, debug build test room, QA lab comedy, clean chunky vector-like shapes, hand-painted texture, bright readable silhouettes, playful system-glitch mood, game-ready asset, no readable text, no UI, no logo, no watermark
```

```text
negative prompt: readable text, letters, subtitles, UI menu, logo, watermark, photorealistic, 3D render, blurry, low resolution, cropped object, horror, gore, distorted anatomy, noisy clutter
```
