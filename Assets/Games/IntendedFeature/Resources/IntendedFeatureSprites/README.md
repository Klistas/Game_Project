# Intended Feature Sprite Slots

이 폴더는 `IntendedFeaturePrototype` 전용 이미지 슬롯입니다.
다른 게임에서 읽지 않도록 `Resources/IntendedFeatureSprites` 경로만 사용합니다.

## 핵심 오브젝트

- `bug_avatar.png` - 플레이어 버그 아바타
- `regression_token.png` - 회귀/증거 토큰
- `ui_token.png` - UI 룸 토큰
- `approval_gate.png` - 승인 게이트
- `tooltip_platform.png` - 실체화된 툴팁 발판
- `patch_note_platform.png` - 패치노트 발판

권장: 모든 오브젝트는 투명 배경 PNG. 지형은 1024x256 또는 1024x512처럼 가로형으로, 캐릭터/토큰은 512x512로 만들면 교체가 쉽습니다.
