# Everyone Innocent Sprite Slots

이 폴더는 `EveryoneInnocentPrototype` 전용 이미지 슬롯입니다.
다른 게임에서 읽지 않도록 `Resources/EveryoneInnocentSprites` 경로만 사용합니다.

## 배경/무대

- `room_backdrop.png` - CCTV 디오라마 룸 배경
- `cctv_frame_top.png` - 상단 CCTV 프레임
- `cctv_frame_bottom.png` - 하단 CCTV 프레임
- `work_table.png` - 증거 작업 테이블
- `display_stand.png` - 복구/전시 스탠드
